
# どの順序でプログラムファイルを使うべきであるか

1. gen_tokens.pl : CLIで別サーバーで操作しながら、手元でブラウザを起動して、2個のトークンを取得。そのうち、リフレッシュトークンは大事。
2. t2a.pl : リフレッシュトークンからアクセストークンを取得する。
